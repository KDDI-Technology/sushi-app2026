### 利用させていただいたリソース提供サイト（敬称略）

- 背景画像→[Adobe Stock](https://stock.adobe.com)
- 効果音(笛の音)→[Springin](https://www.springin.org/sound-stock)
- 効果音(声)→[Web版 VoiceVox](https://www.voicevox.su-shiki.com)
- 効果音(和楽器: 拍子木・小鼓・和太鼓)→[効果音ラボ](https://soundeffect-lab.info/)
- 効果音(上記以外)→[TripArts Music](https://mz4u.net)
